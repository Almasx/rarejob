RareJob Practices PRD

One-liner:
Daily English exercise app. 5 minutes. Every day. Based on RareJob’s Jitsuyo-Eikaiwa curriculum.

Problem:
Users take weekly RareJob lessons but don’t practice between sessions. Skills decay. Progress stalls. They churn.

Solution:
Practices - a daily exercise app. Like Duolingo’s daily streak but using actual RareJob lesson content. User opens app, does 5-min drill, closes. Done.

Target User:
Existing RareJob students who want to reinforce lessons daily without booking more tutor time.

Competitors:
Duolingo: Daily habit works, but content is generic. Not connected to what the user learned with the tutor.
ELSA: Good pronunciation drill, but no curriculum. Random sentences.
RareJob edge: Exercises pulled directly from Jitsuyo-Eikaiwa materials user is studying with their tutor.

The Product

Daily Exercise Flow:
User opens app
Gets 5-min exercise set (listening, speaking, vocab)
User sees score, weak points highlighted
Done. Streak updated.

Exercise Types (from Jitsuyo-Eikaiwa curriculum):
Shadowing: Listen to native audio, repeat. AI scores it
Fill-in-blank: Complete sentences from lesson dialogue.
Role-play: AI plays one role, user speaks the other.
Vocab recall: Flashcard-style review with spaced repetition.
Key Mechanics:
Daily streak (miss a day, lose streak)
5-min cap (respects user time)
Tutor dashboard shows practice stats

Lesson completion +15%

MVP (6 weeks)
Week 1-2: Shadowing exercises for 20 Jitsuyo-Eikaiwa lessons
Week 3-4: PROGOS scoring integration
Week 5-6: Streak system, basic UI polish

Out of Scope
Role-play, vocab recall, tutor dashboard (v2)
