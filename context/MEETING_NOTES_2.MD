# MVP prototype for Jitsuyo e Kaiwa language learning app with RareJob

Fri, 27 Feb 26

### Product Demo & Alignment Discussion

- Presented MVP for language learning companion app
  - Simple onboarding: level selection + interest-based content
  - AI-generated lessons adapted to user interests (travel, sports, etc.)
  - Structured like RareJob lessons: vocabulary, interactive exercises, feedback
- Planned features discussed:
  - AI-generated images based on interests
  - Chatbot assistant for lesson questions
  - Gamification with points/leveling
  - “Send to tutor” functionality for lesson preparation data

### Integration Strategy & Material Relationship

- Clarified app positioning as companion to Jitsuyo e Kaiwa material
  - Will use existing RareJob lesson content as foundation
  - Students practice vocabulary/exercises before tutor sessions
  - Enables tutors to focus on conversation vs. worksheet walkthrough
- Two implementation approaches discussed:
  1. Direct material adaptation: Copy lessons exactly, add visual/interactive elements
  2. Interest-based generation: Use lesson framework but customize with user interests (soccer images for sports fans, etc.)
- Final decision: Companion app using Jitsuyo material as base
  - Students can preview upcoming lessons or review completed ones
  - Same content, different medium (app vs. verbal with tutor)

### Key Features & Student Benefits

- Spaced repetition system for vocabulary retention
  - Critical for A1/A2 level students building vocabulary
  - Data tracking of commonly forgotten words/grammar patterns
- Primary goal: Build student confidence before tutor sessions
  - Address shyness/reluctance to speak during lessons
  - Students practice words beforehand, feel more prepared
- Personalization through data analysis
  - Track individual learning patterns
  - Identify weak areas for targeted practice
  - Complement one-to-many tutoring model with individualized insights
